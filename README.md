# FIA Hackathon – Frontend Assignment

This is a responsive web page created as part of a frontend interview assignment based on a Figma design.

---

## Features

* Animated astronaut section (glow, orbit, floating effect)
* Interactive timeline slider (click, touch, keyboard support)
* Image hover animations
* Fully responsive design (mobile, tablet, desktop)

---

## Tech Stack

* HTML5
* CSS3 / Tailwind CSS
* Vanilla JavaScript

---

## Project Structure

```
index.html
styles.css
script.js
```

---

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser

---

## Notes

* Designed to work on all screen sizes
* Includes basic accessibility support
* Works across modern browsers
